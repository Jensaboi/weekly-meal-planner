# Weekyl meal planner

A web application to create and manage recipes, plan meals, automatic grocery lists and to track calories and nutrition.

## Goals

- Manage recipes
- Meal planning
- Generate gorcery lists
- Calories and nutrition tracking
- Dietary filtering on recipes( no milk, no wheat etc.)
- Shared plans and grocery lists.

## Tech stack

- Next.js
- TypeScript
- PostgresSQL
- Supabase DB
- TailwindCSS
- Shadcn UI

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
